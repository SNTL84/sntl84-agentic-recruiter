## 📋 How to Contribute

### 1. Fork & Clone
```bash
git clone https://github.com/SNTL84/sntl84-agentic-recruiter.git
cd sntl84-agentic-recruiter
```

### 2. Create a Feature Branch
```bash
git checkout -b feature/your-feature-name
```

### 3. Make Your Changes
- Keep code clean and well-commented
- Follow existing file structure
- Test your changes locally by opening the HTML file in a browser

### 4. Commit with a Clear Message
```bash
git commit -m "feat: describe what you added or fixed"
```

### 5. Push & Open a Pull Request
```bash
git push origin feature/your-feature-name
```
Then open a PR on GitHub describing your changes.
