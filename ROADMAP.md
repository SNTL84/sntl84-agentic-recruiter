# 🗺️ Roadmap — SNTL84 Agentic Recruiter

This document outlines the planned features and improvements for future versions.

---

## 🟢 v1.3.0 — Planned (May 2026)

- [ ] **Resume PDF parser** — drag-and-drop PDF → auto-extract candidate data
- [ ] **JD auto-scorer** — paste JD text, get weighted must-have/nice-to-have breakdown
- [ ] **Interview scheduler** — generate calendar invite links per candidate
- [ ] **Candidate comparison view** — side-by-side scoring of top 3

## 🟡 v1.4.0 — Planned (June 2026)

- [ ] **Google Sheets export** — one-click push to Google Sheets
- [ ] **Slack webhook** — notify team when shortlist is ready
- [ ] **Multi-role support** — manage multiple open positions simultaneously
- [ ] **Scoring rubric editor** — customize weights per skill category

## 🔵 v2.0.0 — Vision

- [ ] **AI interview Q&A grader** — evaluate recorded answers via transcript
- [ ] **ATS integration** — Greenhouse, Lever, Workday connectors
- [ ] **Diversity & inclusion metrics** — blind screening mode
- [ ] **Multi-language JD support** — screen candidates across 10+ languages
